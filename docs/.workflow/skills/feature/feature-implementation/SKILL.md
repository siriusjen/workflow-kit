# Feature Implementation

## Overview

Execute one task at a time and leave evidence behind.

## Read First

- `06-上下文包/上下文包-S6-实现.md`
- `03-落地计划/任务清单.json`
- the current T's R and D slices

## Output

- implementation evidence
- task-scoped edits
- `04-实现记录/实现记录-YYYYMMDD-Txx.md`

## Rules

- One T at a time.
- If the task touches code, use a worktree; if it is docs-only, stay in the current branch.
- Record RED/GREEN evidence and never batch multiple T records later.
