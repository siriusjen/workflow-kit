# Feature Intake

## Overview

Turn raw input into a clean feature starting point.

## Read First

- `00-需求输入/`
- any attached source document
- `docs/.workflow/工作流规范.md` 速查卡

## Output

- input summary
- facts
- assumptions
- open questions
- scope boundaries

## Rules

- Keep facts and guesses separate.
- Record missing domain facts as questions or risks.
- Do not skip the input summary even when the source document looks complete.
