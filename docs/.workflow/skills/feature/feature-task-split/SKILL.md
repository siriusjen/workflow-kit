# Feature Task Split

## Overview

Convert a plan into independently executable T tasks.

## Read First

- `03-落地计划/落地计划-v1.md`
- `02-技术方案/技术方案-v1.md`
- `02-技术方案/需求方案映射.json`

## Output

- `03-落地计划/任务清单.json`
- `03-落地计划/RDTV映射表.json`

## Rules

- One T = one coherent change boundary.
- Every task needs scope, dependencies, acceptance, and hours.
- Keep scopes exclusive and minimal.
